# Conto

Personal finance app for tracking virtual accounts, budgets, and bank imports.

Built with Kotlin, Spring Boot, PostgreSQL, and Jetpack Compose.

## Tech Stack

| Layer          | Technology                          |
|----------------|-------------------------------------|
| Backend        | Kotlin, Spring Boot, Flyway         |
| Database       | PostgreSQL                          |
| Frontend       | Android, Jetpack Compose            |
| AI             | Google Gemini API                   |
| Infrastructure | Docker, GitHub Actions              |

## Planned Features

- Virtual accounts and account groups for organizing spending into budget categories
- CSV import for loading transactions from bank exports
- Monthly budget limits with real-time utilization tracking
- AI-powered merchant name translation for cleaning up cryptic bank strings
- Dashboard view showing all balances and budget status

## Status

Work in progress. Currently in the learning and setup phase (Sprint 0).
